# 🚀 Space Exploration Page

A simple webpage built as part of the **Scrimba Frontend Developer Career Path**.  
This project showcases HTML and CSS fundamentals while exploring the theme of space exploration.

---

##  Features
- Clean and responsive layout
- Themed design inspired by space and astronomy
- Sections for exploration, missions, and fun facts
- Easy to customize with your own content

---

## Technologies Used
- **HTML5** for structure
- **CSS3** for styling
- **GitHub Pages / Netlify** for deployment
